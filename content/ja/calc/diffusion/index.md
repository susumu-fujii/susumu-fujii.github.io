プロトンの拡散係数の計算を始める前に、機械学習ポテンシャルと分子動力学法について説明しておきます。
語りが多いですが、是非読んで下さい。

### 機械学習ポテンシャルについて

ここまで順番に進めていれば、第一原理計算について少しイメージが湧いてきているかと思います。
第一原理計算はシュレディンガー方程式を解いて電子分布を決める -> 原子核を動かすというサイクルを繰り返します。
得られる結果は非常に良く実験と一致するので、重宝されています。
ただし、毎回シュレディンガー方程式を解くのに時間がかかります。
また、モデル内の電子数のおよそ3乗に比例して計算コストが大きくなるので、大きなモデルを行うのは一苦労です。

今までは第一原理計算を一つの化合物でやるだけでも十分成果になりましたが、ここ15年ほどで大きく状況が変わりました。
スーパーコンピュータというものに研究者が容易にアクセスできるようになり、たくさんの計算ができるようになりました。
2011年にはアメリカで[Materials Project](https://next-gen.materialsproject.org)というものが発足し、第一原理計算の巨大なデータベースが構築され始めました。
現在では化合物の登録件数は20万を超え、誰でもフリーで使えるためユーザー数も60万人を超えています。
このような大規模第一原理データを使って、計算機上で材料をスクリーニングすることが当たり前になりました。

そしてここ数年で、GPUの性能や機械学習の技術が飛躍的に向上し、より急激に状況が変わりつつあります。
ニューラルネットワークモデルによって、大量の第一原理計算のデータを学習し始めました。
例えば、Metaなどは一億回以上の第一原理計算データを無償で公開しており（[Omat24](https://huggingface.co/datasets/facebook/OMAT24#omat24-dataset)など）、GoogleやMetaなどの大企業や大学がGPUゴリ押しでこれをニューラルネットワークに学習させています。
このような、第一原理計算で求めた原子間相互作用（ポテンシャル）を予測する機械学習モデルのことを、機械学習ポテンシャルと呼びます。
これらの機械学習ポテンシャルも無償で公開されており、日進月歩で最高精度が更新されています（[MatbenchDiscovery](https://matbench-discovery.materialsproject.org)）。
2024年末ごろから性能が急激に上がり、ほとんど第一原理計算と遜色のないレベルに達しています。

特筆すべきところは、このような機械学習ポテンシャルは、第一原理計算と比べて100から10000倍くらい高速なことです。
要するに、今までの100倍から10000倍くらいの規模の計算が実行可能になりました。
今まで第一原理計算では不可能な計算が、手持ちのゲーム用のGPUでも十分に実行できるようになってしまいました。

ただ、第一原理計算も不要になったのではないです。
電子分布やバンド構造、電子伝導特性、磁性などを求めるにはまだ第一原理計算は必須です。
機械学習ポテンシャルは、**まだ**磁性を持つ化合物の再現は苦手です（そのうち解決されそう）。
あくまでも、磁性を持たない化合物のエネルギーや原子間相互作用を求める部分で代用されています。

### 分子動力学法について
分子動力学法は、原子同士に働く力を元にニュートンの運動方程式を解いて、原子のダイナミクス（振動や拡散）を模擬する手法です。
原子にかかっている力を求める -> １ステップ分（例えば1 fs分）原子を動かす、このステップを繰り返すことで、原子の動きを追跡できます。
これを行うと、原子振動（フォノン）による熱伝導や、プロトンやリチウムなどの拡散などを評価することができます。
ただし、この計算から熱伝導度や拡散係数（イオン伝導度）を評価するためには、例えば100万回、モデルの中の原子間の相互作用を求める必要があります。
これを第一原理計算でやろうとすると、100万回シュレディンガー方程式を解く必要があります。
一つのデータのためにそんなに計算リソースを使うのは非現実的です。
なので、より近似した精度の低い原子間ポテンシャルで分子動力学法を行うのが今までは当たり前でした。
精度は低いけど定性的には合っているだろう、何もやらないよりは全然マシ、というノリで僕も使ってました。

ところが機械学習ポテンシャルが開発されたために、第一原理計算と遜色ない精度で分子動力学法を行えるようになりました。
九州大学の玄界のGPUマシンを使えば、3000原子くらいの巨大モデルで、400万ステップの計算が一週間で終わります。
10年前では全く考えられなかった状況です。
今、この道具を使って何を実現するかが、計算材料科学者に問われています。
学生の皆さんは、ここをベースに研究を開始できるので非常に恵まれていると思います。

### プロトンの動きのシミュレーション
ということで、機械学習ポテンシャル分子動力学計算でYb添加BaZrO3のプロトン拡散係数を求めてみましょう。
原子の拡散係数は、ある時間幅での原子の移動距離の二乗（平均二乗変位; MSD）が分かれば求まります。
分子動力学法（MD）をすれば、プロトンの動きが模擬できて移動距離が分かるので、拡散係数も分かるというわけです。
式としては以下の通りです。

$$
\mathrm{MSD}(t)=\left\langle \left| \mathbf{r}(t)-\mathbf{r}(0) \right|^2 \right\rangle
$$

３次元的に原子が拡散する場合、拡散係数$D$は、

$$
D=\lim_{t\to\infty}\frac{1}{6t}\left\langle \left| \mathbf{r}(t)-\mathbf{r}(0) \right|^2 \right\rangle
=\lim_{t\to\infty}\frac{\mathrm{MSD}(t)}{6t}
$$

ここで$r$は原子の座標、$t$は時間です。割と単純です。

第一原理計算を行ったのとは別のGPUマシンで、この計算は行います。
sshでそこにログインして下さい。

```bash
ssh USERNAME@MACHINE
```

例によって、USERNAME、PASSWORD、MACHINEは別で教えます。
ログインできたら、MDというディレクトリを自分の名前にコピーして下さい。

```bash
cp -r MD your_name #your_nameは自分の名字
```

その中に、0600、0800、1000、1200というディレクトリがあると思います。
これは温度に対応していて、それぞれ600 K、800 K、1000 K、1200 Kです。
中を見ると、nvt_init.vaspというのがあります。
これがMDの入力ファイルで、3000原子弱のモデルです。
Samba経由でVESTAで見てみて下さい。第一原理計算では到底扱えないサイズです。
nvt.pyは、機械学習ポテンシャルを利用した分子動力学計算を行うためのPythonコードです。
中身をみてみたら、そんなに複雑じゃないのが分かると思います。

今回の計算では、温度が高くなるほど原子は早く拡散する（熱エネルギーをもらうから）、というのを体感してもらおうと思います。

このマシンにはnvidiaのGPUが2台ついていて、以下のコマンドで、GPUの空き状況がみれます。

nvidia-smi

空いているGPUがあったら、0600、0800、1000、1200のいずれかのディレクトリに入って、以下のコマンドを打てば分子動力学計算を実行できます（ほんとはいくつか予備計算が必要ですが、いきなり本番の計算ができるようにしてあります）。

```bash
CUDA_VISIBLE_DEVICES=0 python nvt.py
```

CUDA_VISIBLE_DEVICESというのは、利用するGPUを選ぶためのもので、0が空いてたら0、1が空いてたら1にして下さい。
放置してたら、MDの途中経過が出力されると思います。
400万ステップは長すぎるので、今回は1万ステップだけです。数十分で終わります。

終わったら、traj.extxyzというファイルが生成されていることを確認して下さい。
この中に、10ステップごとの原子位置が保存されているので、これを可視化すれば原子の動きが見れます。
可視化するためのソフトとして、[Ovito](https://www.ovito.org)をインストールして下さい。
インストール後、traj.extxyzをドラッグアンドドロップすれば、アニメーションで見れると思います。
（読み込み直後はなんだか原子が大きすぎたりして見辛いと思いますが、工夫すれば美しくできます。聞いて下さい。）

とりあえず、温度で全然動き方が違うというのを実感してもらえればと思います。
そしてそのアニメーションは、0.5 fs x 10000ステップ、つまりわずか$5 \times 10^{-12}$ sの間に起きた出来事になります。
皆さんが作った試料の中でも、ナノスケールではこんな現象が起きているわけです。

### 拡散計算の算出

計算が終わったら、平均二乗変位と拡散係数を見てみましょう。
といっても、１万ステップくらいじゃ全然精度が足りないので、正確には求められないことに注意して下さい。
以下のコマンドをMDディレクトリで実行してみてください。

```bash
python get_msd.py
```

0600/traj.extxyz等を読み込んで拡散係数を表示するとともに、MSDの図をプロットしてくれます。
Samba経由で、fig_msd.pngを開いて見てください。
MSDが直線になっていると思いますが、この傾きが拡散係数になります。
温度によって拡散係数が大幅に上昇することが、定量的に分かるかと思います。

このように、我々が測定可能な物理量は、ミクロな世界で決定されています。
なので、正しい（妥当な）仮定やモデリングを行えば、原子・電子レベルで材料特性を調べたり、優れた機能が発現するメカニズムを解明することが可能です。
仮想的な組成で安定性や特性を調べれば、計算機の中で材料探索することも出来ます。
計算、実験、機械学習を組み合わせて研究を進めることは、現代的な材料科学で顕著な成果を出すための必須事項と言えます。
